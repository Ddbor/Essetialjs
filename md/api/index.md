# 使用方式

## import

```js
import { isString } from 'essetialjs'

isString('abc')
```

## require

```js
const ESS = require('essetialjs')

ESS.isString('hello world')
```

## Global

```html
<script src="essetialjs.min.js"></script>
<script>
  ESS.isString('abc')
</script>
```
