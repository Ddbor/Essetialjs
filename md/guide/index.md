# 安装

## NPM

```bash
npm install essetialjs
```

## Yarn

```bash
yarn add essetialjs
```

## PNPM

```bash
npm add essetialjs
```

## HTML 中引入

下载 [`essetialjs.min.js`](https://github.com/Ddbor/Essetialjs/tree/main/dist/essetialjs.min.js)

```html
<script src="essetialjs.min.js"></script>
```
